# 广东省公考/公职类招聘官方源头清单（2026-06-01）

本页整理广东省级与 21 个地市的公考/公职类招聘官方发布源头，覆盖公务员省考、选调生、事业单位集中招聘、三支一扶、教师、医疗卫生、辅警/社区等。

> 使用原则：培训机构、聚合站、本地生活网站只适合发现线索；报名、资格审查、岗位要求、考试安排均以官方公告、附件岗位表和报名系统为准。

## 一、监控优先级

1. **省级总入口**：广东组织工作网、广东省人社厅、广东人事考试网、广东省公务员/选调/事业单位/三支一扶报名系统。
2. **地市人社/考试入口**：市人社局、政府门户人事信息、人事考试栏目。
3. **专项主管部门**：教师看教育局，医疗看卫健局，辅警看公安局，社工/社区看民政局、人社局和区县政府。
4. **区县/单位官网**：直属学校、直属医院、街镇、区县单位招聘通常分散发布，需要按目标城市继续下钻。

- 区县级教师、医疗、辅警、社区工作者招聘发布较分散，需要按目标城市继续下钻。
- 广东国企招聘没有统一省级公考入口，应另行按广东省国资委、各市国资委和各集团有限公司官网整理。
- 个别报名系统或年度专题 URL 可能随年份调整，应在每年公告发布后复核。
