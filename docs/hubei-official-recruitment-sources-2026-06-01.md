# 湖北省公考/公职类招聘官方源头清单（2026-06-01）

本页整理湖北省级与全省 17 个市州/直管市/神农架林区的公考/公职类招聘官方发布源头，覆盖公务员省考、选调/遴选、事业单位统招、省直事业单位、三支一扶、教师、医疗卫生、辅警、社区等。

> 使用原则：培训机构、聚合站、本地信息站只适合发现线索；报名、资格审查、岗位要求、考试安排均以官方公告、附件岗位表和报名系统为准。

## 一、监控优先级

1. **省级总入口**：湖北省人事考试网、湖北省人社厅省直事业单位专题、省级考试报名/准考证/成绩入口。
2. **地市人社/考试入口**：市州人社局、人事考试院/考试网、政府门户“招考录用/人事信息/公考公招”。
3. **专项主管部门**：教师看教育局，医疗看卫健委，辅警看公安局，社区/社工看民政局、社会工作部、人社局和区县政府。
4. **技术报名系统**：`pzhl.net` 等非政府域名系统只有在官方公告明确指向时，才作为官方指定报名技术系统使用。

## 四、未完全穷尽的部分

- 区县级教师、医疗、辅警、社区工作者招聘发布较分散，需要按目标城市继续下钻。
- 湖北国企招聘没有统一公考入口，应另行按湖北省国资委、市州国资委和各集团有限公司官网整理。
- 个别地市人事考试网站存在证书异常或非 `.gov.cn` 技术域名，报名时应从官方公告链接进入并二次核验。
- 湖北省委组织部、湖北省公务员局是否存在长期稳定独立专题页，还可后续专项补查；当前已确认省级核心入口为湖北省人事考试网及省人社厅专题。
